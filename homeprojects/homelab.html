Home Lab

I have a couple Raspberry Pis and a NUC that I use as servers to play around with.

Homebridge

I use homebridge to "smart-ify" my home. Homebridge allows me to use non-Apple compatible devices and functions with my iPhone/iPad.

For example, I have control over IR/RF devices, Wake on LAN support, and a Zigbee dongle for any Zigbee compatible devices.

Pi-Hole

Pi-Hole provides adblocking on my home network.

Plex

My NUC hosts a Plex server so I can stream movies and TV shows throughout the house.

SyncThing

I use SyncThing to synchronize my PC, laptop, NUC, SteamDeck, and iPhone. I can set up specific directories to sync between them.

Website

This site started on a Raspberry Pi NGINX server with HTML, CSS, Javascript, and PHP. I moved it to Github for free and more secure hosting.
