PC Setup

Between multiple windows installs and updates, I got tired of manually setting up my PCs over and over. I ended up setting up a folder that is synced via cloud storage (iCloud at the moment) and using SyncThing for LAN. In the folder is a number of scripts to...

  • Remove bloat
  • Quickly configure system settings
  • Download applications I always use
  • Backup / restore app and extension settings
  • Updating installed applications

These scripts include .bat files, powershell commands / .ps1 files, 'winget' commands, and settings from %appdata% and other folders for apps.


Remove Bloat

I started with some .bat files from the internet, then removed / added some of my own tweaks to uninstall the windows app I (and probably most people) never use.

Configure System

I have a couple .bat files: one that requires admin access, one that doesn't. They both go through system and user settings that I usually change, like setting my preferred mouse pointer, or turning off 'enhanced pointer precision' and it asks if I want to change each setting one-by-one. Usually, I'm going to do all of them, but this at least gives me some control over it, just in case. Each action is tied to a powershell file to update the commands.

Download Apps

I'm a big fan of using winget to install applications from the windows terminal. I made a few .bat files to run through different types of apps that I usually install. Like the system configurations, these ask one-by-one.

I recently re-wrote this to also have the option to queue up downloads, so you can quickly queue several up and install them all at once at the end of the list.

App Settings

This one isn't as technical. It's simply a folder structure to store settings for individual applications and extensions. I include instructions in a text file for myself and a shortcut to the settings folder, usually in %appdata%.

Some day, when I'm feeling ambitious, I'm planning to set up an auto-backup system for this, but I need to work out how to best handle it.

Update Apps

This .bat file just does a winget command to see what applications have updates available. It also has a list of commands I can type to update all or individual items, so I don't have to go look it up all the time.
