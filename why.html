Example 1

Just a few days before a big election, our team had to update software and manually create custom usernames and passwords for hundreds of laptops delivered to polling locations one-by-one. I wrote a batch file in Excel to automatically create custom usernames and passwords on laptops based on the computer name, along with copying applicable software over, vastly speeding up the process and cutting down on input errors. Since we were already using flash drives to copy the software over, we instead ran the batch script and it did the rest for us.

Example 2

Our election iPad software only allowed us to download logs for one device at a time. Each log file had the same name. So I used a program called Autohotkey and an Excel doc to download each one after the election and rename them automatically, changing a daunting manual process into a button click.
